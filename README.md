# Shea-Note-Taker

## Description 

This is a tool for organized note taking. Notes are kept on a server in a json file, which can be added and fetched in order to keep organized.  

## Built With
* Javascript
* Express.js
* Node.js
* ES6
* HTML
* CSS
* JQuery
